# PidTorClient

PidTorClient — Online-модуль Lampac-NG для работы с торрентами, **использующий TorrServer, настроенный на устройстве пользователя**.  
Lampac только выполняет поиск раздач и не взаимодействует с TorrServer.

## Установка

1. Скопируйте папку `PidTorClient` в `/lampac/mods/`
2. Перезапустите Lampac

## Как это работает

1. Lampac получает список раздач через API JacRed.
2. Клиентский плагин автоматически добавляет к запросу параметр `pidtor=1`.
3. После выбора раздачи magnet-ссылка передаётся в TorrServer напрямую с клиента.
4. Воспроизведение запускается через стандартный плеер Lampa.

## Настройка клиента Lampa

1. Убедитесь, что в Lampa настроен TorrServer (основной или дополнительный).
2. Установите плагин:
   - Lampa → Настройки → Расширения → Добавить плагин
   - URL: `http://ваш_сервер:9118/pidtor-client.js`
  
После установки плагина источник автоматически использует TorrServer, указанный в настройках Lampa.  
Lampac не взаимодействует с TorrServer напрямую — все операции по добавлению magnet-ссылок выполняются на стороне клиента.

Если плагин не установлен или TorrServer не настроен, источник покажет соответствующее предупреждение.

## Настройки в init.conf

```
"PidTorClient": {
  "enable": true,                    // Включить/выключить модуль
  "displayname": "PidTor Client",    // Отображаемое имя источника
  "displayindex": 552,               // Порядок отображения в списке источников
  "group": 0,                        // Группа доступа (0 - без ограничений)
  "group_hide": false,               // Скрыть для групп

  "redapi": "https://jac.red",       // Адрес API JacRed (например: https://jac.red или http://192.168.1.100:9117)
  "apikey": "",                      // API ключ (для jac.red не требуется, для других: "myapikey123")

  "min_sid": 20,                     // Минимальное количество сидов (0 - показывать все)
  "max_size": 0,                     // Максимальный размер раздачи в байтах (0 - без ограничений, 2147483648 = 2GB)
  "max_serial_size": 0,              // Максимальный размер для сериалов (0 - без ограничений)
  "forceAll": false,                 // Не фильтровать раздачи по качеству (true - показывать 480p и ниже)
  "emptyVoice": true,                // Показывать раздачи без озвучки (false - только с озвучкой)
  "sort": "sid",                     // Сортировка: "size" - по размеру, "sid" - по сидам, "date" - по дате
  "filter": "",                      // Фильтр по названию или озвучке (Regex, например: "LostFilm|HDrezka")
  "filter_ignore": "",               // Игнорировать раздачи (Regex, например: "CAMRip|TS")

  "msg_no_plugin": "⚠️ Плагин не установлен",            // Сообщение при отсутствии плагина
  "msg_no_plugin_desc": "Установите плагин PidTorClient и настройте TorrServer",  // Описание
  "msg_no_torrserver": "⚠️ TorrServer недоступен",        // Сообщение при недоступности TorrServer
  "msg_no_torrserver_desc": "Проверьте настройки TorrServer в приложении"  // Описание
}
```

## Примечания

- Для работы необходим клиентский плагин `pidtor-client.js`. Он встроен в модуль и автоматически доступен по адресу `http://ваш_сервер:9118/pidtor-client.js`.
- TorrServer должен быть настроен в приложении Lampa.
- При отсутствии плагина источник отображает предупреждение.
- Поддерживаются основной и дополнительный TorrServer, настроенные в Lampa.
- Кэш поиска — 20 минут. Для мгновенного обновления перезайдите в карточку фильма.
